# Lab 01 – Active Directory

## Objective
Set up a basic Active Directory environment using Windows Server 2022  
and connect a Windows 10 client to simulate a company network.

## Lab Setup
- DC-01 (Windows Server 2022)
  - 2 vCPUs / 4GB ram
  - Bridged Network

- Client-01 (Windows 10)
  - 2 vCPUs / 4GB ram
  - Bridged Network
